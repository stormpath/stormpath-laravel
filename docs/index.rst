Stormpath-Laravel's documentation
=================================
Stormpath-Laravel is a package for Laravel and Lumen that makes it simple to add users and user data to your
application. It aims to completely abstract away all user registration, login, authentication, and authorization
problems, and make building secure websites painless. And the best part? You don’t even need a database!

This documentation will show you how to get started with Stormpath-Laravel. If you’re a new Stormpath-Laravel
user, start here!

Contributing
------------
We welcome anyone to make contributions to this project. Just fork the `develop`_ branch of our `GitHub Repository`_,
make your changes, then issue a pull request on the `develop`_ branch.

Any pull request you make will need to have associated tests with them. If a test is not provided, the pull
request will be closed and not merged. Also, any pull requests made to a branch other than `develop`_ will
be closed and a new submission will need to be made to the `develop`_ branch.
